# PLANS

- Active: currently being executed.
- Completed: accepted and finished with date.
- Tech Debt: known debt with impact and proposed resolution path.

## Operating Rules

- New initiative: create plan file in `active/` and add an index link here.
- Completion: move file from `active/` to `completed/`, keep filename stable, update index.
- Deferred debt: track in `tech-debt/` with impact, owner, and trigger condition.
