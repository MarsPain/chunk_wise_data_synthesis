# FRONTEND

This repository is backend and CLI focused. There is no browser frontend in the current scope.

## Current UI Surface

- CLI scripts under `scripts/`
- Python package APIs under `src/`

## If Frontend Scope Is Added

Before adding any web UI or app UI:
- Create a product spec in `docs/product-specs/`.
- Add a dedicated design doc in `docs/design-docs/`.
- Update `docs/DESIGN.md` and `AGENTS.md` map links.
- Add tests and validation rules for frontend-specific docs if they become required.
