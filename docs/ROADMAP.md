# ROADMAP

## Milestone M1: Context System Hardening

- Keep `AGENTS.md` concise as repository map.
- Move durable knowledge to `docs/` as canonical record.
- Enforce docs structure and link integrity through executable validation.

## Milestone M2: Plan Lifecycle Operations

- Keep plan states current across `active`, `completed`, and `tech-debt` buckets.
- Require plan index updates in `docs/PLANS.md` when state transitions happen.

## Milestone M3: Governance in CI

- Integrate docs validator into default CI test command.
- Fail pull requests when docs structure or links regress.
